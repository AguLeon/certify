### Certify
Web development project for a yelp-like web app for car listings


## Main features
- Listings creation
- Image uploads
- Interactive, geoclustered map
- Search filters
- User authentication and authorization
- User review system
- Deployment
- Implemented as webpage and also as a RESTful API

## Main tools
- Node.js
- ExpressJS
- MongoDB
- Cloudinary
- Deployed in Heroky
